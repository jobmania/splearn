# Splearn 개발가이드

## 아키텍처
- 헥사고날 아키텍처
- 도메인 모델 패턴


### 계층
- Domain Layer
- Application Layer
- Adapter Layer

>  외부(Actor) -> 어댑터 -> 어플리케이션 -> 도메인 

## 패키지 
- domain 
- application
  - required (어플리케이션이 필요로 하는 기능들 [메일발송, 외부 api])
  - provided (어플리케이션이 제공하는 기능 )
- adapter
  - webapi 
  - persistence
  - integration
  - security
